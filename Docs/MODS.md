# Mods

This repository is specifically for developing a game called Bound. I'm defining mods as equivalent to games. The `GameEngine` directory handles a bunch of setup and backend functionality to make a simple 2D game in SDL2.

## Bound

A skill-based, 2D puzzle-action game where the player must navigate a character through a series of deadly traps and timed obstacles often involving precise movement, timing, and avoidance of explosive triggers. Refer to the genre of maps in Starcraft called Bound for more information.
